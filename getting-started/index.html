    Apa itu Turunan Numerik?¶

    Turunan Numerik adalah turunan yang menentukan hampiran nilai turunan fungsi f (diberikan dalam bentuk tabel). Meskipun menggunakan metode numerik untuk menghitung turunan fungsi yang tersedia, akan tetapi perhitungan turunan dapat mungkin dihindari. Dikarenakan, nilai turunan numerik umumnya kurang teliti dibandingkan dengan nilai fungsinya.

    Menghitung Turunan Numerik¶

    Misal diberikan nilai-nilai x di x0 - h, x0, dan x0 + h, serta nilai fungsi untuk nilainilai x tersebut. Titik-titik yang dapat diperoleh adalah (x-1, f-1), (x0, f0), dan (x1, f1), yang dalam hal ini x-1 = x0 - h dan x1 = x0 + h.

    Terdapat 3 pendekatan dalam menghitung nilai f '(x0):

    1. Hampiran selisih-maju (forward difference approximation)

      $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}+h\right) -f\left( x_{0}\right) }{h} = \dfrac{f_{1}-f_{0}}{h} $$

      Tafsiran geometri dari Hampiran selisih-maju dapat dilihat seperti gambar dibawah: ​

    2. Hampiran selisih-mundur (backward difference approximation)

      $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}\right) -f\left( x_{0}-h\right) }{h} = \dfrac{f_{0}-f_{1}}{h} $$

      Tafsiran geometri dari Hampiran selisih-mundur dapat dilihat seperti gambar dibawah: ​

    3. Hampiran selisih-pusat (central difference approximation)

      $$ f'\left( x_{0}\right) = \dfrac{f\left( x_{0}+h\right) -f\left( x_{0}-h\right) }{2h} = \dfrac{f_{1}-f_{-1}}{2h} $$

      Tafsiran geometri dari Hampiran selisih-pusat dapat dilihat seperti gambar dibawah: ​

    Rumus-rumus turunan numerik dari ketiga pendekatan diatas dapat diturunkan dengan 2 cara, yaitu:

    1. Dengan bantuan deret Taylor

    2. Dengan hampiran polinom interpolasi

    Kedua cara tersebut menghasilkan rumus yang sama

    Rumus-Rumus Turunan¶

    1. Rumus Turunan Pertama¶

    (Selisih-maju) $$ f_{0}^{'}=\dfrac{f_{1}-f_{0}}{h}+O\left( h\right) $$

    (Selisih-mundur) $$ f_{0}^{'}=\dfrac{f_{0}-f_{-1}}{h}+O\left( h\right) $$

    (Selisih-pusat) $$ f_{0}^{'}=\dfrac{f_{1}-f_{-1}}{h}+O\left( h^{2}\right) $$

    (Selisih-maju) $$ f_{0}^{'}=\dfrac{-3f_{0}+4f_{1}-f_{2}}{2h}+O\left( h^{2}\right) $$

    (Selisih-pusat) $$ f_{0}^{'}=\dfrac{-f_{2}+8f_{1}-8f_{-1}+f_{-2}}{2h}+O\left( h^{4}\right) $$

    2. Rumus Turunan Kedua¶

    (Selisih-pusat)

    \[ f_{0}^{"}=\dfrac{f_{1}-2f_{0}+f_{-1}}{h^{2}}+O\left( h^{2}\right) \]

    (Selisih-mundur)

    \[ f_{0}^{"}=\dfrac{f_{-2}-2f_{-1}+f_{0}}{h^{2}}+O\left( h\right) \]

    (Selisih-maju)

    \[ f_{0}^{"}=\dfrac{f_{2}-2f_{1}+f_{0}}{h}^{2}+O\left( h\right) \]

    (Selisih-maju) $$ f_{0}^{"}=\dfrac{-f_{3}+4f_{2}-5f_{1}+2f_{0}}{12h}+O\left( h^{2}\right) $$

    (Selisih-pusat)

    \[ f_{0}^{"}=\dfrac{-f_{2}+16f_{1}-30f_{0}+16f_{-1}-f_{-2}}{12h}^{2}+O\left( h^{4}\right) \]

    3. Rumus Turunan Ketiga¶

    (Selisih-maju)

    \[ f_{0}^{'''}=\dfrac{f_{3}-3f_{2}+3f_{1}-f_{0}}{h^{3}}+O\left( h\right) \]

    (Selisih-pusat)

    \[ f_{0}^{'''}=\dfrac{f_{2}-2f_{1}+2f_{-1}-f_{-2}}{h^{3}}+O\left( h^{2}\right) \]

    4. Rumus Turunan Keempat¶

    (Selisih-maju)

    \[ f_{0}^{iv}=\dfrac{f_{4}-4f_{3}+6f_{2}-4f_{1}+f_{0}}{h^{4}}+O\left( h\right) \]

    (Selisih-pusat)

    \[ f_{0}^{iv}=\dfrac{f_{2}-4f_{1}+6f_{0}-4f_{-1}+f_{-2}}{h^{4}}+O\left( h^{2}\right) \]

    Code¶

    Berikut merupakan Algoritma Code dari Metode Turunan dengan deret Taylor:

    #include<math.h>
#include<stdio.h>

double factorial(int num){
    double fact =1;
    int i;

    for(i=1;i<=num;i++){
        fact = fact*i;
    }
    return fact;
}

int main(){
    double tempt, x,sum=1.0;
    int term,counter;
    int  sign =-1.0;
    printf("Masukkan nomor istilah sampai anda ingin menghitung nilainya\n");
    scanf("%d",&term);
    printf("\nMasukkan nilai x\n");
    scanf("%lf",&x);
    tempt=x;

    //ex
    for(counter=1;counter <= term ; counter++){
        sum = sum + ((pow(x,counter))/(factorial(counter)));
    }

    printf("\nJadi deret Taylor dari e^x di sekitar %.1lf adalah %.15lf \n",tempt,sum);

    //sinx
    x = x * 3.14159265359 / 180;
    sum = x;

    for(counter=3;counter<=term;counter+=2){
        sum = sum + (sign * (pow(x,counter))/factorial(counter));
    }

    printf("Deret Taylor dari sin(x) di sekitar %.1lf adalah %.15lf \n",tempt,sum);

    // cosx
    sum = 1.0;
    sign = -1.0;
    for(counter=2;counter <= term ; counter += 2){
        sum = sum + (sign *(pow(x,counter))/factorial(counter));
        sign = sign * (-1);
    }

    printf("Deret Taylor dari cos(x) di sekitar %.1lf adalah %.15lf \n",tempt,sum);
}
