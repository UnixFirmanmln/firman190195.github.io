    Apa itu Persamaan Differensial?¶

    Persamaan diferensial merupakan suatu persoalan matematis yang sering dijumpai dalam bidang teknik lingkungan. Sering kali suatu persamaan diferensial tersebut tidak dapat diselesaikan secara analitik sehingga diperlukan suatu metode numerik untuk menyelesaikannya.

    Metode Euler¶

    Metode Euler merupakan metode yang paling sederhana dari persamaan differensial, metode ini merupakan turunkan dari deret Taylor. Penyelesaian initial value problems menggunakan metode ini dilakukan melalui Persamaan berikut:

    \[ y_{i}+1=y_{i}+f\left( X_{i},Y_{i}\right) h \]

    dimana i merupakan tahapan iterasi.

    Algoritma Metode Euler¶

    1. Tentukan titik awal integrasi x0 dan y0
    2. Tentukan jumlah iterasi n dan step size h yang digunakan
    3. Lakukan integrasi menggunakan Persamaan diatas

    Algoritma tersebut, selanjutnya dapat disusun ke dalam sebuah fungsi R. Dimana fungsi tersebut adalah sebagai berikut:

    euler <- function(f, x0, y0, h, n){
  x <- x0
  y <- y0

  for(i in 1:n){
    y0 <- y0 + h*f(x0, y0)
    x0 <- x0 + h
    x <- c(x,x0)
    y <- c(y, y0)
  }

  return(data.frame(x=x, y=y))
}

    Contoh: Selesaikan persamaan di bawah ini, jika diketahui f(0)=1 menggunakan h=0,05 dan n=100!

    \[ f^{1}\left( x,y\right) =\dfrac{x}{2x+1} \]

    Jawab: Penyelesaian secara analitik dari persamaan tersebut untuk nilai f(0) = 1 sebagai berikut:

    \[ f\left( x\right) =\sqrt{2x+1} \]

    Secara numerik persamaan tersebut dapat diselesaikan sebagai berikut:

    Iterasi ke-1

    \[ Y\left( 0,5\right) =1+0,5\times \dfrac{1}{\left( 2.0\right) +1}=1.05 \]

    Iterasi ke-2

    \[ Y\left( 0,1\right) =1,05 + 0,05\times \dfrac{1}{\left( 2.0,05\right) +1}=1.097727 \]

    Kita dapat juga menggunakan fungsi euler() untuk menyelesaikan persamaan diatas secara numerik. Hasil yang diperoleh selanjutnya dapat diplotkan dengan hasil yang diperoleh menggunakan metode analitik. Berikut adalah sintaks yang digunakan:

    # metode numerik
f1 <- function(x,y){y/(2*x+1)}
num <- euler(f1, x0=0, y0=1, h=0.05, n=100)

# metode analitik
f2 <- function(x){sqrt(2*x+1)}
x0 <- 0
y0 <- 1
x <- x0
y <- y0

for(i in 1:100){
  y0 <- f2(x0+0.05)
  x0 <- x0+0.05
  x <- c(x, x0)
  y <- c(y, y0)
}
true <- data.frame(x=x, y=y)

    Berdasarkan hasil visualisasi diatas dapat dilihat bahwa menggunakan metode Euler dapat dengan baik memberikan pendekatan nilai integrasi persamaan.

    Metode Heun¶

    Metode Heun ini merupakan salah satu peningkatan dari metode Euler yang melibatkan 2 buah persamaan.

    Persamaan pertama disebut sebagai persamaan prediktor maksudnya yaitu persamaan yang digunakan untuk memprediksi nilai integrasi awal.

    \[ Y_{i+1}^{0}=Y_{i}+f\left( X_{i},Y_{i}\right) ,h \]

    Persamaan kedua disebut sebagai persamaan korektor yang merupakan persamaan mengoreksi hasil integrasi awal.

    \[ Y_{i+1}+1=y_{i}+\dfrac{f\left( x_{i},y_{i}\right) +( x_{i}+1,y_{i+1}^{0}}{2}h \]

    Metode Heun pada Chapter ini merupakan metode prediktor-korektor satu tahapan. Akurasi integrasi dapat ditingkatkan dengan melakukan koreksi ulang terhadap nilai koreksi semula menggunakan persamaan kedua.

    Algoritma Metode Heun¶

    1. Tentukan titik awal integrasi x0 dan y0
    2. Tentukan jumlah iterasi n dan step size h yang digunakan
    3. Lakukan prediksi nilai awal dengan Persamaan (pertama)
    4. Lakukan koreksi nilai awal menggunakan Persamaan (kedua)
    5. Lakukan koreksi terhadap nilai koreksi yang dihasilkan dari sebelumnya menggunakan Persamaan (kedua)

    Berikut merupakan sintaks yang digunakan untuk membangun sebuah fungsi yang dapat melakukan proses integrasi menggunakan metode Heun:

    heun <- function(f, x0, y0, h, n, iter=1){
  x <- x0
  y <- y0

  for(i in 1:n){
    ypred0 <- f(x0,y0)
    ypred1 <- y0 + h*ypred0
    ypred2 <- f(x0+h,ypred1)
    ykor <- y0 + h*(ypred0+ypred2)/2
    if(iter!=1){
      for(i in 1:iter){
        ykor <- y0 + h*(ypred0+f(x0+h,ykor))/2
      }
    }
    y0 <- ykor
    x0 <- x0 + h
    x <- c(x, x0)
    y <- c(y, y0)
  }

  return(data.frame(x=x,y=y))
}

    Code program penyelesaian ODE¶

    import numpy as np
from scipy.integrate import odeint
import matplotlib.pyplot as plt

# fungsi yang mereturn dy/dt
def model(y,t):
    k = 0.3
    dydt = -k * y
    return dydt

# kondisi awal
y0 = 5

# point waktu
t = np.linspace(0,20)

# penyelesaian ODE
y = odeint(model,y0,t)

# hasil plot
plt.plot(t,y)
plt.xlabel('time')
plt.ylabel('y(t)')
plt.show()
